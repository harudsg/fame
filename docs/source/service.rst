- 이메일: support@sktelecom.com
- 홈페이지: [fame.sktelecom.com](https://fame.sktelecom.com)

문의 내용을 바탕으로, 홈페이지 [fame.sktelecom.com](https://fame.sktelecom.com)에 접속하여 개인 계정을 생성해주세요. 여기서 서비스 정보를 등록하고 계정을 설정할 수 있습니다.
